# Recipe Conversion Calculator
This is a simple tool meant for converting recipes.

# Example

## 1. Ingredients
454, grams, Flour

3, cups, Water

22, litres, yield
## 2. Choose how to scale
- Scale by ingredient
> ex. 44, yield
- Scale by scaling factor
> ex. To halve a recipe scale by 0.5
## 3. Calculate
You should now have your converted recipe
